
欢迎使用appAdmin后台管理系统，下面是对本项目实现技术的介绍。

架构{
	后台{
		Spring：众所周知
		SpringMvc：众所周知
		Jpa：众所周知（底层支持多种 orm 框架，本项目采用 hibernate ）
		
		Apache Shiro：这是一个强大易用的 Java 安全框架，提供了认证、授权、加密和会话管理功能，可为任何应用提供安全保障-从命令行应用、移动应用到大型网络及企业应用。
		Jcaptcha：这是一个开源的用来生成图形验证码的 Java 开源组件，使用起来也是非常的简单方便。
		Jackson：这是一个 Java 用来处理 JSON 格式数据的类库，性能非常好。
		C3P0：这是一个开源的JDBC连接池，它实现了数据源和JNDI绑定，支持JDBC3规范和JDBC2的标准扩展。目前使用它的开源项目有Hibernate，Spring等。
		存储：插件化开发，只需要您点一点，进行一个配置，马上可以更换您的存储方式，不需要开发哦：{
			ftp存储。
			本地存储。
			阿里云oss存储。
		}
		缓存：{
			memorycache：提供了memorycache的整合，提供工具类，方便使用。
			ehcache：提供了ehcache缓存机制，当前 shiro 使用您的缓存就是ehcahe的。
		}
		系统设置：{
			系统设置由于是一个高度可配置的存在，为了方便操作，系统设置不持久化到数据库，而是持久化到xml文件，非常容易理解的哦。
		}
	
	}
	视图{
		freemarker：java web开发中很常见的一种视图模板技术，相比传统的jsp有很多很多的优点。
	}
	服务器{
		当前使用嵌入式jetty服务器，方便、快捷、运行极速。
	}
}
